Folder: GSE116256_RAW
This folder contains the raw single-cell data (GSE116256) used for this project.

Due to GitHub file size limitations, raw files are hosted externally:
Download link:
https://drive.google.com/drive/folders/1h_0RxDzszHCcR5fWLr_7AxGgZp6EhDhc?usp=drive_link

To reproduce the analysis:
1. Download all files from the link above.
2. Place them under this folder (`AML_OCT_v2/GSE116256_RAW/`).
3. Run the R script from the project root.
